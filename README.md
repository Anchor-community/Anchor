## Description

A Discord Bot for [Anchor](https://anchor.institute/) Guild.

## Installation

```
$ yarn install
```

## Running the app
```
# dev
$ yarn debug

# production
$ yarn compile
$ yarn start
```

## Test

```
$ yarn test
```
